## 🚀 Featured Projects

### [Eyes-on-Orbit](https://github.com/MoamenMohameed/Eyes-on-Orbit)
Real-time satellite tracking and visualization using TLE data and SGP4 orbit propagation (via Satellite.js), with 3D rendering through CesiumJS. Includes coordinate transformations (ECI → ECEF → geodetic) and ground footprint/coverage calculations.

### [osrm-routing](https://github.com/MoamenMohameed/osrm-routing)
Python tool for calculating road routes between two coordinates using the OSRM routing engine, returning route geometry and turn-by-turn directions.
